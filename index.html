Bower

Bower

Пакетный менеджер для веба.

Добавим jQuery без bower

  1. Заходим на сайт jQuery
  2. Скачиваем jQuery
  3. Добавляем jQuery в проект

Добавим jQuery с bower

$ bower install jquery

Установка

Для работы нужны Node и npm и Git

$ npm install -g bower

Папка bower_components

$ bower install jquery 

			project
			|_bower_components
				|_jquery
					|_dist
					|_src
					|_...

main

$ bower list --paths 
			// jquery: 'bower_components/jquery/dist/jquery.js'

Компоненты можно коммитить

$ bower install jquery normalize-css ... 

			project
			|_bower_components
				|_jquery/
				|_normalize-css/
				|_...
				|_...
				|_...

А можно не коммитить

bower.json

			"dependencies": {
			  "name": "version",
			  "name": "version"
			}

bower install

Создаем bower.json

$ bower init

--save

$ bower install --save jquery normalize-css ...

bower.json 
			"dependencies": {
			  "jquery": "~2.1.1",
			  "normalize-css": "~3.0.2"
			}

Отличие от npm

У npm древовидная структура зависимостей

Пример npm

A > C

B > C

npm install

			node_modules
			|_A
			| |_C
			|_B
			  |_C

Пример bower

A > C B > C bower install bower_components |_A |_B |_C
